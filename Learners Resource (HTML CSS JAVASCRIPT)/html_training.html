Em tags make content italic but hold more meaning than i tags.

Links / Hyperlinks

Links allow us to travel between places within and outside a page.

Using the a tag we can make some text we want to be our clickable link text.

Within the a tag created, use the href attribute which requires a full URL to act as our link.

Youtube

URL stands for Uniform Resource Locator.

Many pages have same site navigation at the top of their page. These pages are individual html files all in the same folder.

Example Page

Larger websites are sorted by section into separate pages.

For ease of set up, many of these pages are set up with a template for easy replication and editing.

Email Links and mailto:

Within the href attribute you can link an email, be sure to add "mailto:" before the email is placed, like "mailto:ServiceDesk@example.org"

Target attribute

Some links can open a new tab or have effects similar to that.

As seen above we have a link to youtube that opens in a new tab. That is because after the href attribute leading to the URL of youtube, we have another attribute; target. Target is set to = "_blank" for opening a new tab when transfering to a new URL.
